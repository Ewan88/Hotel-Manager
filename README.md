
## A Vue.js and MongoDB web application

### Server

Install server dependencies:

```
cd server
npm install
```

Run a mongoDB server:

```
mongod
```

Seed the database:

```
npm run seeds
```

Run express:

```
npm run server:dev
```

### Client

Project setup

```
cd ../client
npm install
```

Compiles and hot-reloads for development

```
npm run serve
```
